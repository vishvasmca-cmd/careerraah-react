# **App Name**: BlogFlow

## Core Features:

- Blog Post Fetching: Fetch and display blog posts from the existing blog site (ensure the existing blog exposes an API for data retrieval or uses a headless CMS approach).
- Content Presentation: Render fetched content using React components.
- Search Functionality: Implement search functionality allowing users to search and filter blog posts based on keywords.
- AI-Powered Summarization Tool: Integrate AI-powered tool that offers brief summaries for posts within the listing to enhance content discoverability.
- AI Assessment Tool: Use AI to conduct assessment for students and parents.

## Style Guidelines:

- Primary color: Deep purple (#673AB7), chosen for its blend of stability and creativity, which suits a blogging platform. It contrasts well on a light background and provides a modern yet trustworthy feel.
- Background color: Light lavender (#EDE7F6), desaturated hue of purple to offer a gentle and unobtrusive backdrop.
- Accent color: Soft blue (#3F51B5), a hue analogous to the primary purple but distinct enough in brightness and saturation to draw attention to key interactive elements without overwhelming the visual experience.
- Headline font: 'Belleza', sans-serif with personality; body font: 'Alegreya', a serif to promote readability.
- Simple line icons for categories and navigation.
- Clean and responsive grid layout for blog posts.
- Subtle fade-in effects when loading content.
